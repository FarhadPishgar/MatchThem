## What's New

### Version 0.8.2

This is a spit and polish update to improve the documentation and to implement compatibility for weighted analyses (from the [`survey`](https://cran.r-project.org/package=survet) package) and for new matching and weighting methods.

### Version 0.8.1

This is the first release of the [`MatchThem`](https://cran.r-project.org/package=MatchThem) package on the [Github](https://github.com/FarhadPishgar/MatchThem) and [Comprehensive R Archive Network (CRAN)](https://cran.r-project.org/package=MatchThem).
